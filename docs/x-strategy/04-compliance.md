# 04. 著作権・コンプライアンス(根本原因 #5)

## 結論(電験の過去問)

一般財団法人 電気技術者試験センターが公表している過去問題は、
**教育目的の範囲内なら許諾・使用料は不要**で問題集・テキストに利用できる。
ただし **条件あり**:

1. **出典を明記**(年度・期・試験区分)
2. **問題の一部を改変している場合は「改題」とその旨を明記**
3. 試験制度の意義に反しない教育目的であること

> 出典: [電気技術者試験センター 問題と解答](https://www.shiken.or.jp/chief/second/qa/) /
> [日本著作権教育研究会](https://www.jcea.info/nyushi2.html)
> ※利用前に公式の「利用上の留意点」を一次情報として必ず確認すること。

つまり **「係数いじり」は `○年度 二種 一次 改題` と書けば正攻法**。

---

## 出典フッターテンプレ(画像内＆本文に常時表示)

**完全オリジナル問題**
```
出典: DENKEN-OS オリジナル問題（CC表記は配布方針に従う）
```

**過去問の改題**
```
出典: 令和○年度 第二種電気主任技術者試験 一次試験 機械（一部改変・改題）
```

**過去問をそのまま引用**
```
出典: 令和○年度 第二種電気主任技術者試験 二次試験 電力・管理
```

- 画像の隅に出典表記を**テンプレ化**(毎回手間なく明記)。

---

## オリジナル比率の方針

- **完全オリジナル問題を主軸**にする(検算エンジンで生成)。
  → 著作権フリー＋模倣されにくい＋差別化を同時達成。
- 改題は「本試験の傾向を示す」目的で限定的に。依存しない。

---

## AI生成物の権利・規約

- 生成に使うAI(Claude API 等)の **利用規約で出力の再利用可否・商用利用** を確認。
- 過去問をAIに入力する際、入力データの扱い(サーバ保存等)を確認。
- 生成問題の著作権の所在、**ユーザー投稿問題(UGC)の権利処理**を
  アプリの利用規約に明記(将来の事業化で効く)。

---

## やらないこと(NG)

- 他社問題集(オーム社等)の問題の**転載・改題**(別の著作物。参照はOK、複製はNG)。
- 市販解説の**引き写し**(解説文・図は自作する。手書き図は信頼にもなる)。
- 出典・改題の明記**漏れ**(怠ると教育目的の例外から外れる)。

---

## 公開する「品質・出典方針」雛形

アプリ/プロフに掲載し、透明性で信頼と通報リスク低減を両取りする。

```
【DENKEN-OS の問題ポリシー】
・問題は完全オリジナル、または公表過去問の改題です
・改題・引用は出典（年度・期・区分）を必ず明記します
・全問を検算し、合格者の監修を受けています
・誤りのご指摘は歓迎し、最速で訂正・公開します
```
