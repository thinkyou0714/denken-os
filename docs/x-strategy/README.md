# DENKEN-OS / X 情報発信 実装パッケージ

「電験二種、一緒に受けませんか?」── *凡人が成り上がるサクセスストーリー* として
X(旧Twitter)で発信し、学習アプリ DENKEN-OS の成長エンジンにするための運用一式。

これは「アイデア集」ではなく **そのまま運用できる実装** です。各ファイルはコピペ可能な
テンプレ・チェックリスト・SOP(標準作業手順)で構成されています。

---

## 0. 戦略の一行サマリ

> 「解説者(権威)」ではなく **「当事者の記録 × 道具づくり × 共闘」** で立つ。
> アプリは無料で配って仲間を集め、課金は信頼が溜まってから。

成長ループ:

```
① 発信で集める  →  ② Aha体験で掴む  →  ③ 継続で定着  →  ④ シェアで増幅  →（①へ）
   物語/今日の一問     登録直後に弱点判明    ストリーク/作業会   学習記録カード/招待
```

## 4. 絶対原則(これだけは破らない)

1. **間違った問題・著作権違反を絶対に出さない** — 1問の事故が全てを壊す(`03`/`04`)。
2. **続けられる量しかやらない** — 毎日全部やろうとして潰れるのが最大の失敗(`08`)。
3. **課金は信頼の後** — 未合格期は無料で価値提供。物語の山場(合格)で回収(`07`)。
4. **堀はコミュニティと物語** — 技術は真似される。人と物語は移せない(`06`)。
