# 検証済みサンプル問題(品質パイプラインの実演)

`03-quality-pipeline.md` の5ステップを通した一例。**完全オリジナル＝著作権フリー**で、
答えが綺麗な値(力率0.8 / 3.2kW)になる係数だけを採用している。

---

## 問題(人間向け)

> **No. T-0001 ／ 難易度 ★★☆☆☆ ／ 二種一次・理論 ／ 三相交流電力**
>
> 平衡三相 Y(スター)結線の負荷に、線間電圧 200 V の対称三相交流電圧を加えた。
> 1相あたりのインピーダンスが **Z = 8 + j6 〔Ω〕** のとき、
> この負荷が消費する **三相有効電力 P 〔kW〕** はいくらか。
>
> ① 2.56　② 3.2　③ 4.0　④ 9.6

**正解: ②(3.2 kW)**

---

## 検算(ステップ4「人の目」を明示)

```
|Z| = √(8² + 6²) = √100 = 10 Ω
力率 cosφ = R/|Z| = 8/10 = 0.8
相電圧(Y結線) V_p = 200/√3 V
相電流(=線電流) I = V_p/|Z| = (200/√3)/10 = 20/√3 A

三相有効電力:
  P = 3·I²·R = 3·(20/√3)²·8 = 3·(400/3)·8 = 3200 W = 3.2 kW

別解で再検算(独立な式で一致を確認):
  P = √3·V_l·I_l·cosφ = √3·200·(20/√3)·0.8 = 3200 W  ✓ 一致
  S = √3·V_l·I_l = 4000 VA → P = S·cosφ = 4000·0.8 = 3200 W ✓
```

### 選択肢の引っ掛け(すべて「成立する誤り」)
| 選択肢 | 値 | 対応する典型ミス |
|---|---|---|
| ① | 2.56 kW | 力率を二重に掛けた(3I²R にさらに cosφ) |
| ② | **3.2 kW** | **正解** |
| ③ | 4.0 kW | 力率を掛け忘れ、皮相電力 S と混同 |
| ④ | 9.6 kW | Y結線で √3 を忘れ、線間電圧を相電圧として計算 |

→ 最頻誤答は **④(√3忘れ)**。解答編で「ここを見落とすと④」と解説する。

## 無限生成のやり方(この問題から類題を作る)

- `R`, `X` を `realistic_range` 内で振り、**|Z|・力率・Pが綺麗になる組だけ採用**。
  例: (R,X)=(6,8)→|Z|=10,力率0.6 / (R,X)=(3,4)→|Z|=5,力率0.6。
- 線間電圧を 200→400→6600 と変える。
- 「Y結線 ↔ Δ結線」で √3 の扱いを変えた解法違いの類題に展開。
- いずれも **必ず検算→チェックリスト(03)** を通してから出題する。
