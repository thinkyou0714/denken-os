# 05. アルゴリズム適合 ＆ エンゲージメント SOP

## 2026 X アルゴリズム 5つの絶対法則

```
① 会話(リプ往復)が王 ── リプの往復は「いいね」の150倍の価値
② ブックマーク(保存)が最強シグナル ── いいね以上。"後で見返す実用性"
③ 滞在時間(dwell) ── 長く読ませる投稿が伸びる(スレッド/画像/動画)
④ 初速30分 ── 投稿後30分の反応でフォロワー外への拡散が決まる
⑤ 外部リンク本文直貼りはインプ激減 ── リンクはリプ欄/プロフへ逃がす
```
> 2025年11月〜 X は Grok 主導の AI 型アルゴリズムに移行。「量から質」へ。
> **フォロワーが少なくても、専門性×会話の質があればインフルエンサー級の露出が可能**。
> 無名スタートに有利な地形。

---

## 投稿ルール(毎回守る)

| 項目 | ルール |
|---|---|
| リンク | 本文に貼らない。「詳細はプロフ」or リプ欄に置く |
| ハッシュタグ | 関連3個以内(#電験二種 #今日の一問 等)。トレンド乱用はスパム判定 |
| メディア | 毎投稿に画像/動画/スレッドのいずれかを添える(テキストのみは評価低) |
| 投稿時間 | 受験生の活動帯(早朝/通勤/夜21時前後)。データで最適化 |
| 問いかけ | 末尾に具体的な問い(「あなたはどっち派?」)。リプを誘発 |
| エンゲベイト | 「いいねして」の連呼はNG。価値で自然に取る |

---

## リプ営業 SOP(無名期の唯一の突破口・毎朝30分)

```
1. 大手電験アカウント(電験王・Taira 等)の最新投稿に、的確な補足リプ
   → 相手のフォロワーに自分が露出する最速の手段
2. 同期(同時期スタートの受験仲間)5〜10人の投稿に絡む
   → 互いの初速30分をブーストする体制（自然な相互応援の範囲で）
3. #電験二種 検索で困っている人に答える
   → 感謝とフォローを生む
4. 来たリプには必ず返す(質問で返して会話を続ける形)
   → 会話ツリーの長さ＝アルゴリズム評価
```
- スタンスは **「奪う」より「与える」**。反論より補強。敵を作らない。
- リプは「ひと言」で終えず、**往復が生まれる**返し方を意識。

---

## 会話を伸ばす投稿の型

- **問いで終える**(「これ、引っ掛けとして成立してる?」)。
- 投稿後、**自分で補足リプをぶら下げる**(滞在時間とツリーの厚みを作る)。
- 議論を呼ぶ問題(★4-5)を意図的に混ぜる。

## 保存を取りに行く型

- 公式まとめ・暗記ゴロ・**図解1枚で完結**するコンテンツ(リンク不要＝流出ペナルティなし)。
- 「保存して後で解いてね」と自然に促す。

---

## 初速30分の作り方

- 出題直後は**解答予想リプに即返信**して会話を自分で育てる。
- 事前に「初速を作る同期」を持っておく(②参照)。
- 投稿時間を最適化し、フォロワーがオンラインな時間に出す。
