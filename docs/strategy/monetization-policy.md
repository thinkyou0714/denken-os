# 収益化ポリシー（公開方針書）

DENKEN-OS の収益化に関する約束と基準の正本（SSOT）。
手順書は [`monetization-setup.md`](monetization-setup.md)、案の全カタログは
[`ideas/17-bridge-revenue-100.md`](ideas/17-bridge-revenue-100.md)。
（17-D4/D10/D18/D20/D21/D22/D23 を統合）

## 1. 無料ユーザー体験の保護原則（フリーミアム憲章）— 17-D22

1. **復習（FSRS）・復習ドリル・間違いノート・進捗・公式集・質問タブは無料で使い続けられる。**
2. 無料枠（学習タブの新しい問題の1日上限）を狭める場合は、**事前に告知**してから行う。
3. キルスイッチ（`MONETIZATION.enabled=false`）で**全ゲートを即時解除できる状態**を常に維持する。
4. 収益導線は非侵襲原則に従う: 能動的な案内はアプリ全体で**1日1件まで**・
   **学習中（出題〜採点）には出さない**・**設定でオフにできる**。
5. 進行中の模試など「開始済みの学習セッション」を、課金状態の変化で中断・破棄させない。

## 2. 収益源の一覧と開示方法

| 収益源 | 対価性 | アプリ内での開示 |
|---|---|---|
| Pro ライセンス（買い切り/期限付き） | あり（役務: キー発行＋公式ビルドの便利機能） | ペイウォールに無料範囲・買い切りを明示 |
| 寄付・応援（Ko-fi 等） | **なし**（見返りを約束しない） | 「機能は変わりません」を明記 |
| アフィリエイト（教材ガイド等） | 第三者広告 | リンク近傍に「PR・アフィリエイト」バッジ必須 |
| note・BOOTH（記事・PDF） | あり（コンテンツ） | 外部サイトである旨を表示 |
| スポンサー（将来） | 広告 | 「スポンサー」表記必須（下記基準） |

## 3. 寄付と Pro の区分 — 17-D4

- 寄付は**無対価の応援**。ライセンスキーは渡さない（渡すと販売＝特商法・消費税の扱いが変わる）。
- Pro は**対価のある販売**。特商法表記・返金特約の対象。
- 「寄付したら Pro がもらえるか」→ もらえません。Pro が欲しい方は購入導線から。
- 高額の寄付に対する返礼は行わない（対価性の混同を避ける）。

## 4. アフィリエイト・広告の基準 — 17-D10/D16

- **開示**: 2023年10月施行の景表法ステマ規制に従い、アフィリエイトリンクの近傍に
  「PR・アフィリエイト」表示を必ず添える（アプリはこれを部品で強制する）。
- **編集独立性**: レビュー・教材ガイドの結論は金銭で変えない。悪い点も書く。
- **受けない広告**: 情報商材・高額塾・合格保証を謳うもの・出所不明の教材。
- **正確性**: 電卓は試験規定（一般電卓のみ可・関数電卓不可）の解説を必ず併記する。
- **Pro 機能の中に第三者広告は出さない**（有料ユーザーへの広告は信頼毀損）。

## 5. 返金・キャンセル — 17-D18

- デジタルライセンスキーの性質上、**キー発行後の返金は原則不可**（返品特約として表示）。
- 例外: 二重購入・誤購入直後・キーが技術的に機能しない場合は、購入記録の確認のうえ
  **返金＋当該キーの無効化（返金済みリスト運用）**で対応する。
- 鍵ペアの再生成は既発行キーを全滅させるため、個別対応を原則とする。

## 6. OSS（MIT）と課金の両立 — 17-D20

- コードは MIT。**フォークして自前ビルドすれば全機能を無料で使えます**。これは仕様であり、隠しません。
- Pro が買っているのは「公式ビルドの利便性＋署名キーという役務＋開発継続への応援」です。
- したがって「Pro でしか使えない」とは表現しない（正確には「公式版では Pro 機能」）。
- 回避は可能です。それでも応援してくれる人に支えられる設計（honor system）を選びます。

## 7. 問題データ（CC-BY-SA-4.0）の商用整理 — 17-D21

- `data/` の自作問題・解説は CC-BY-SA-4.0。**第三者も帰属表示＋同一条件で商用利用可**
  （塾・競合アプリの利用も設計上許容済み）。
- 運営者自身は原著作者なのでライセンス制約なく商用利用できる。
- 過去問の原文は（一財）電気技術者試験センターの著作物であり、CC-BY-SA を付せるのは
  自作の解説・改題部分のみ。**有償頒布物への過去問転載は許諾が必要**。
- 将来、新規データのみ別ライセンスに変更することは可能（遡及はしない）。

## 8. スポンサー掲載規約（将来用・17-D5/D7）

- 掲載は「スポンサー」表記＋ロゴ/名称のみ。問題・解説・レビューの内容には一切関与させない。
- 支援者クレジット掲載は本人の同意を得た希望者のみ。辞退・削除請求にいつでも応じる。

## 9. 計測とプライバシー

- 導線の表示/クリックは**端末内 localStorage に整数カウントのみ**。個人識別子・端末 ID なし。
- 外部への自動送信は一切しない。書き出しはユーザー操作による手動コピーのみ。
- 流入 UTM は初回1件だけ端末内に記録し、URL からは即座に除去する。
