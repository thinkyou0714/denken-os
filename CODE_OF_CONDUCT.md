# 行動規範 (Code of Conduct)

## 私たちの約束

DENKEN-OS は「凡人が当事者として挑む」コミュニティです。年齢・経験・合否に関わらず、
誰もが安心して学び・質問し・間違えられる場であることを最優先にします。

## 期待される振る舞い

- 初学者の質問・間違いを歓迎する（「弱い主人公」を笑わない）。
- 誤りの指摘は敬意をもって。指摘してくれた人にはクレジットと感謝を。
- 出典・著作権を尊重する。他者の解説や問題を無断で引き写さない。
- 学習以外の論争・誹謗中傷・スパム・宣伝目的の行為をしない。

## 許容されない振る舞い

- ハラスメント、差別、攻撃的・侮辱的な言動。
- 個人情報の無断公開。
- 故意に誤った情報を広める行為。

## 対応

問題のある言動を見かけた、または受けた場合は、リポジトリの Issue もしくはメンテナへの
連絡（GitHub）で報告してください。メンテナは内容を確認し、警告・投稿削除・参加制限などの
適切な対応を行います。

本規範は [Contributor Covenant](https://www.contributor-covenant.org) の精神に基づきます。
